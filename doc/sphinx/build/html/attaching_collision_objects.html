Attaching collision objectsÂ¶

If other objects are going to collide with the hiar, you need to attach them as collision objects.

  1. Select:

    • the polygon mesh(es) for the object(s) to collide with the hair, and
    • the wmFigaro node for the hair.

  2. From the Figaro menu, select Add selected meshes as collision objects.

    Figaro creates a wmFigCollisionNode. This lets you control the attributes of the collision object.
