Animating the rodsÂ¶

To animate the rods, you need to use the wmBarbershop guide curve deformer.

  1. Select:
    • the wmFigRodNode, and
    • the wmFurSetNode.
  2. Run the script wmBarberShop -gcd.

The rods and hair will now animate in sync.
